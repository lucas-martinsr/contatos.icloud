# Estruturação de Base de Contatos (SQL | Data Cleaning | ETL)

Projeto desenvolvido para estruturação e padronização de uma base de contatos proveniente de múltiplas fontes (iCloud e WhatsApp), com foco na utilização para campanhas e geração de leads.

*Os dados apresentados neste projeto foram anonimizados para garantir a privacidade das informações.*

## Objetivo
Criar uma base de dados limpa, padronizada e estruturada a partir de mais de 11.000 contatos, possibilitando seu uso em campanhas de marketing e análise de dados.

## Etapas do Projeto (ETL)
1. Extração de dados provenientes de iCloud e WhatsApp
2. Transformação e limpeza dos dados com SQL
3. Padronização de formatos (nome, telefone e DDD)
4. Identificação e remoção de registros duplicados
5. Enriquecimento dos dados com associação de DDD → cidade
6. Estruturação da base final para análise e uso comercial

## Técnicas Aplicadas
- Data Cleaning
- Deduplicação de dados
- Padronização de dados
- Enriquecimento de dados
- Estruturação de base para análise

## Resultados
- Base de dados consolidada e padronizada
- Melhoria na qualidade e organização das informações
- Redução de inconsistências e duplicidades
- Base pronta para utilização em campanhas e análise de leads

## Tecnologias Utilizadas
**SQL** → manipulação, limpeza e transformação de dados
**GitHub** → versionamento e documentação
